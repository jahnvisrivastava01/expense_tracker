# 💸 Expense Tracker (React)

## 🚀 Overview

A clean and interactive Expense Tracker built using React.

Users can add expenses, filter them by category, calculate total spending, and persist data using localStorage.

---

## ✨ Features

* ➕ Add expenses (amount + category)
* ❌ Delete expenses
* 📊 Total calculation (auto updates)
* 🔍 Filter by category (Food, Transport, etc.)
* 💾 Data persistence using localStorage
* 🎨 Clean and modern UI (Tailwind CSS)

---

## 🧠 Tech Stack

* React (useState, useEffect)
* JavaScript (ES6)
* Tailwind CSS

---

## ▶️ How to Run

```bash
npm install
npm run dev
```
